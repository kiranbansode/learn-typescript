# What Is TypeScript & Why Should You Use It

## What Is TypeScript

1. A JavaScript Superset(a language building up on JavaScript)
2. Add new features + advantages to JavaScript
3. Browser can't execute TypeScript
4. Typescript compiles to normal JavaScript
5. Features are compiled to JS 'workarounds', possible errors are thrown

## Why Should You Use It

1. Catch unwanted behavior at runtime by using types
